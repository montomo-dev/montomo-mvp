# 種族(tribe)一覧

DQ風の「種族」分類。エンブレム/アイコン制作用の一覧です。
各SPECIESエントリの `tribe` フィールドで管理しています(js/data/monsters.js)。

## tribeキー一覧

| キー(英字ID) | 日本語名 | 英語名 |
|---|---|---|
| kemono | けもの | Beast |
| dragon | ドラゴン | Dragon |
| bird | とり | Bird |
| bug | むし | Bug |
| plant | しょくぶつ | Plant |
| aqua | すいせい | Aqua |
